Multi-Agent Playground visual simulator

Inspired by TensorFlow Playground. Configure agents, tools, topology and a task — watch the system execute and see cost / latency / quality estimates. No real API calls — this is a deterministic simulator that visualizes how multi-agent patterns behave.

🔑 API Keys (saved in your browser only)
Stored in localStorage on your machine. Never sent anywhere except directly to api.anthropic.com / api.openai.com. Clear keys
How live mode works
Each node in the graph is a real LLM call to the provider you picked, using your API key. The browser sends with anthropic-dangerous-direct-browser-access: true for Claude. Output of one agent becomes input to the next, following the topology. Cost is computed from real usage; latency is wall-clock; quality is the human assessment of the final output (your call).
